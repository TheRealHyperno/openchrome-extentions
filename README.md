These extensions can and will have bugs and may require extra tweaking on your bot to get right.

As of June 20 2026 these are the following types of files extensions for each type of thing

Addons: .hypernoaddon

Plugins: .hypernoplugin

Plugin Pack: .hypernopack

Context/Behavioral: .hypernoskill
